# Scent-jo-plugin

> 개인 프로젝트 Claude 플러그인

## Features

- lighthouse skill
- design-system-extractor skill

## Installation plugin

```bash
claude plugin add --git github.com/pcjo1202/scent-jo-plugin

```

## Install Skills

```bash
claude plugin add --git github.com/pcjo1202/scent-jo-plugin/skills/lighthouse
claude plugin add --git github.com/pcjo1202/scent-jo-plugin/skills/design-system-extractor
```

```bash
unzip {skill-file} -d ~/.claude/skills/
```
